# Project 3 Final

by Antonio Gutierrez


# Overview

This project is a revision of Project One, which was a portfolio that contained artists that inspired me, lists of my favorite things, and an about me page. The first version of this site gave me a lot of stress during the construction due to being unfamiliar with HTML and CSS, but as I worked on various assignments through the semester I gained more confidence in my ability to code and apply constructive criticism into my design.

# Notable Changes
- A new home page with a responsive media query
- A new GIF that functions as the branding for the website
- GIF is now in the header of the site and visible on every page
- New hover effect on the header and footer that creates a white box around it and changes the text color to black
- New text captions for every entry of the list instead of one that summarizes each list
- Headers at the top of each page changed to "Goldman" font
- A text typing animation for the header of each page upon loading the page 
- Adjusted line lengths and font sizes on each page
- Increased margins between various elements on inspirations and top fives page
- Hover effect on all images that increases size by 10%
